# Images Directory

This directory contains static images used throughout the Fast Food App.

## Structure

Organized by purpose:
- `logos/` - Restaurant and app logos
- `menu/` - Food and dish images
- `icons/` - UI icons and graphics
- `backgrounds/` - Background images

## Image Guidelines

- Use optimized images (compressed, appropriate size)
- Preferred formats: WebP, PNG, JPEG, SVG
- Include alt text for accessibility
- Use descriptive filenames

## Current Status

Currently using Font Awesome for icons and external image sources.
Add restaurant-specific images here as needed.
